# 📖 Project Overview

India has become one of the world's fastest-growing startup ecosystems, attracting billions in investments across fintech, edtech, healthtech, e-commerce, AI, and SaaS.

This project explores startup funding data to understand how investments are distributed across industries, cities, investors, and funding stages.

Using Python for data cleaning, exploratory data analysis (EDA), and visualization, the project identifies patterns that can support investment decisions, market research, and business strategy.

---

# 🎯 Business Problem

Investors, founders, and product teams often ask questions like:

- Which industries receive the highest funding?
- Which cities attract the most startup investments?
- Who are the most active investors?
- How has startup funding changed over time?
- Which sectors are emerging as investment hotspots?

This project answers these questions using exploratory data analysis.
